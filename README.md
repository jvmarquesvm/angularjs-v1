######## OBS na Chamada da API #############
No arquivo previsao-tempo-controller.js na chamada de um API Rest deve ser informado um hash.

Esse hash é fornecido pelo dono da API - https://openweathermap.org/
Para tanto, é necessário fazer um cadastro e adquirir este hash e substituir <<<IDTEMPO>>> pelo hash adquirido.
